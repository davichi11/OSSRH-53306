# spring-boot-starter-redisson
spring-boot集成Redisson
=====================
本模块是为了在Spring Boot整合Redisson,方便配置尔来,只需引入本模块并配置好Redis即可.
### 如何使用
1. 首先需要配置spring的Redis,exp:
`spring.redis.host=127.0.0.1`
1. 引入本依赖
2. 注入RedissonClient即可使用,无需额外配置
